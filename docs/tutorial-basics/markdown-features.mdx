# 마크다운 기능

Docusaurus는 **[마크다운](https://daringfireball.net/projects/markdown/syntax)**과 몇 가지 **추가 기능**을 지원합니다.

## Front Matter

마크다운 문서는 상단에 [Front Matter](https://jekyllrb.com/docs/front-matter/)라는 메타데이터를 가지고 있습니다:

```text title="my-doc.md"
// highlight-start
---
id: my-doc-id
title: My document title
description: My document description
slug: /my-custom-url
---
// highlight-end

## 마크다운 제목

[링크](./hello.md)가 있는 마크다운 텍스트
```

## 링크

일반 마크다운 링크가 지원되며, URL 경로나 상대 파일 경로를 사용할 수 있습니다.

```md
[페이지 생성](/create-a-page) 방법을 알아봅시다.
```

```md
[페이지 생성](./create-a-page.md) 방법을 알아봅시다.
```

**결과:** [페이지 생성](./create-a-page.md) 방법을 알아봅시다.

## 이미지

일반 마크다운 이미지가 지원됩니다.

정적 디렉토리(`static/img/docusaurus.png`)의 이미지를 참조하기 위해 절대 경로를 사용할 수 있습니다:

```md
![Docusaurus 로고](/img/docusaurus.png)
```

![Docusaurus 로고](/img/docusaurus.png)

현재 파일을 기준으로 상대 경로로 이미지를 참조할 수도 있습니다. 이는 마크다운 파일과 함께 사용할 이미지를 가까운 곳에 배치하는 데 특히 유용합니다:

```md
![Docusaurus 로고](./img/docusaurus.png)
```

## 코드 블록

구문 강조가 지원되는 마크다운 코드 블록을 사용할 수 있습니다.

````md
```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Hello, Docusaurus!</h1>;
}
```
````

```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Hello, Docusaurus!</h1>;
}
```

## 주의사항 (Admonitions)

Docusaurus는 주의사항과 콜아웃을 만들기 위한 특별한 구문을 제공합니다:

```md
:::tip 팁

이 멋진 기능 옵션을 사용하세요

:::

:::danger 주의

이 작업은 위험합니다

:::
```

:::tip 팁

이 멋진 기능 옵션을 사용하세요

:::

:::danger 주의

이 작업은 위험합니다

:::

## MDX와 React 컴포넌트

[MDX](https://mdxjs.com/)는 문서를 더욱 **인터랙티브**하게 만들고 **마크다운 내에서 React 컴포넌트**를 사용할 수 있게 해줍니다:

```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '20px',
      color: '#fff',
      padding: '10px',
      cursor: 'pointer',
    }}
    onClick={() => {
      alert(`라벨 ${children}과 함께 색상 ${color}를 클릭했습니다`)
    }}>
    {children}
  </span>
);

이것은 <Highlight color="#25c2a0">Docusaurus 그린</Highlight> 입니다!

이것은 <Highlight color="#1877F2">Facebook 블루</Highlight> 입니다!
```
